使用Github

  • Terminal 指令列 + VSCode ，或用
  • Github Desktop
  • 先把 專案 clone 到自己的電腦（資料夾命名不要有中文）
  • 嘗試建立自己的資料夾或頁面，請盡情隨意嘗試修改
  • 把修改或建立的東西 push 到 Github 共用的專案上，讓改變可以在 模板頁面 這邊被找到
  • 如果已經能夠 pull / commit / push 來操作 github ，可以試試看把自己的頁面套上公用的模板，可以參考下圖，拿 index.html 去改。

模板使用說明

同學們可以複製這頁 index.html 然後把你們要開始嘗試切版的內容，放在以下註解包含的部分。後續更新模板可以持續沿用。

檔名與資料夾命名規則

  • html, css, js 檔案名稱都用小寫
  • 可以用的字元是英文字母 a~z 與數字 0~9 中間可以用減號 "-" 和底線 "_" 。
  • 盡量不要太長，以清楚表示為主
